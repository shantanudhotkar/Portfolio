// 03. projects

Featured Projects

Remote Terminal Unit (Industrial IoT Gateway)

Industrial IoT edge node on STM32F401CCU6 acquiring 16 digital I/O via MCP23S17 SPI expander, 4–20 mA analog inputs on 12-bit ADC, and energy parameters from a Schneider EM6436H meter via RS485 Modbus RTU (8E1, CRC-16) at 1-second cycle. All data is published as JSON telemetry to HiveMQ cloud via ESP8266 using a custom bare-metal MQTT library (CONNECT/PUBLISH/PINGREQ) written entirely from scratch.

STM32F401CCU6 ESP8266 RS485 Modbus RTU MQTT MCP23S17 Bare Metal

Smart Room Monitor with FreeRTOS

FreeRTOS v202406 manually ported bare-metal on STM32F401CCU6 — no CubeMX — with custom Cortex-M4 SysTick, PendSV, and SVCall handlers. A 4-task concurrent system reads DHT11 temperature/humidity and HC-SR04 ultrasonic distance using DWT bit-bang drivers. Sensor data shared via SensorMsg_t queue, I2C/UART buses protected with dual Mutexes, and LCD alerts triggered via a 3-bit Event Group on threshold breaches.

STM32F401CCU6 FreeRTOS DHT11 HC-SR04 I2C LCD Event Groups Mutex

Energy Monitor with RS485 & Telegram

Real-time energy monitoring using ESP32 and PZEM-004T sensor over RS485 with MAX485 IC for half-duplex UART-to-RS485 conversion. Implements Modbus RTU to acquire voltage, current, active power, energy, frequency, and power factor at ±0.5% accuracy on a 15-second cycle. Sends live telemetry and threshold-based alert notifications directly to mobile via Telegram Bot API over Wi-Fi.

ESP32 PZEM-004T RS485 MAX485 Modbus RTU Telegram Bot Wi-Fi
